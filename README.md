# *トレログ*(https://kukku999.xsrv.jp/)

・トレーニング内容（名前、重量、回数）を記録、管理するWebアプリです
・その日に行うトレ部位をカレンダーに登録できます
・他のユーザーが行ったトレーニングを共有して見て、イイね！することができる。
